# Детектор физической активности (Android)

[cite_start]Приложение для автоматического определения типа физической активности пользователя (Покой, Ходьба, Бег) на основе анализа данных с акселерометра в реальном времени[cite: 13, 15].

## 📋 Описание проекта
[cite_start]Приложение разработано в рамках лабораторной работы №1 по дисциплине «Разработка мобильных приложений»[cite: 4, 5]. [cite_start]Оно классифицирует активность пользователя, используя математическую обработку показаний датчика ускорения, и предоставляет визуальное и звуковое подтверждение смены состояний[cite: 13].

## ✨ Основные возможности
* [cite_start]**Детекция активности**: Автоматическое определение состояний «Покой», «Ходьба» и «Бег»[cite: 15, 35].
* [cite_start]**Визуальный интерфейс**: Отображение текущего статуса на главном экране приложения[cite: 17, 42].
* [cite_start]**Звуковое сопровождение**: Воспроизведение различных аудиосигналов при смене типа движения[cite: 18, 36].
* [cite_start]**Энергоэффективность**: Регистрация слушателя сенсора происходит в `onResume()`, а отписка — в `onPause()`, что экономит заряд батареи при сворачивании приложения[cite: 19, 48].

## 🛠 Технологии и инструменты
* [cite_start]**Язык**: Java [cite: 27]
* [cite_start]**Платформа**: Android SDK [cite: 26]
* [cite_start]**Датчики**: `SensorManager` (Accelerometer) для получения данных об ускорении[cite: 28].
* [cite_start]**Мультимедиа**: `MediaPlayer` для обработки звуков[cite: 29].
* [cite_start]**Потоки**: `Handler` для периодического анализа накопленных данных[cite: 30].

## 📐 Алгоритм обработки данных
1.  [cite_start]**Сбор данных**: При каждом изменении данных датчика вычисляется ускорение по формуле корня из суммы квадратов осей X, Y и Z[cite: 32].
2.  [cite_start]**Фильтрация**: Для подавления шумов применяется фильтр низких частот: `ускорение = 0.9 * предыдущее + дельта`[cite: 33].
3.  [cite_start]**Анализ**: Каждые 2 секунды проверяется счетчик движений, превысивших порог[cite: 35]:
    * **< 5 движений**: Покой.
    * **5 — 25 движений**: Ходьба.
    * **> 25 движений**: Бег.
